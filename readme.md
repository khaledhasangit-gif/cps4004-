Northshore Logistics Ltd - Centralised Database SystemProject OverviewThis project involves the design and implementation of a robust, centralised database solution for Northshore Logistics Ltd, a fictional medium-sized distribution company. The system aims to replace fragmented paper-based and spreadsheet systems with an integrated solution to manage daily shipments, vehicle schedules, driver routes, and customer order information.Technical SpecificationsThe system is developed using the following tools and environment:Language: Python 3.11+ Database: SQLite 3 Libraries: * sqlite3: For database development tkinter: For the graphical user interface pandas: For data handling hashlib & secrets: For secure password hashing and authentication tokens logging: For tracking system activities datetime: For managing schedules and delivery dates Key Features1. Shipment and Delivery ManagementAdd and update shipment records, including sender/receiver details and item descriptions.Track incident reports such as delays, damaged goods, or failed deliveries.Manage financial details, including transportation costs and payment statuses.Generate status reports (e.g., In Transit, Delivered, Returned).2. Inventory and Fleet ManagementMonitor stock levels and reorder points across multiple warehouse locations.Manage vehicle information, including capacity and maintenance schedules.Maintain driver records, license information, and shift assignments.3. Data SecurityRole-Based Access Control (RBAC): Restricts access based on user roles (e.g., Admin, Manager, Driver, Warehouse Staff).Data Encryption: Protects sensitive information like customer addresses and payment records.Auditing: Logs database activities to detect unauthorised modifications or data misuse.Submission RequirementsThe final submission consists of two primary components:Software Artefact (Zip File): Contains the source code, the SQLite database, and a full Git Log demonstrating version control history with regular commits.Written Report (PDF): An 1800-word document covering the design, implementation, analysis, and evaluation of the system
